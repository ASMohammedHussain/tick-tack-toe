How to run the Gesture Tic Tac Toe app with webcam access:

1. In Windows Explorer, open this folder:
   c:\Users\loq\OneDrive\Documents\tick tack toe

2. Double-click start-server.bat

3. Open your browser and go to:
   http://localhost:8000

4. Click Start Camera on the webpage.

5. Allow the browser to use your webcam when prompted.

Important:
- The app must be served from localhost or HTTPS.
- Browsers usually block webcams for pages opened with file://.
- If camera access is denied, refresh the page and allow permission again.
