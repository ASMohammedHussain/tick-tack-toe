Place your exported Teachable Machine image model files here.

The website expects the following files in this folder:
- model.json
- metadata.json
- weights.bin (and any supporting binary files)

For the gesture mapping to work best, train class labels such as:
- one, two, three, four, five, six, seven, eight, nine
- or cell 1, cell 2, ..., cell 9

Then open index.html in a browser that supports webcam access.
